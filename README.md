# 🚀 Banking Transaction Analytics System

An end-to-end Banking Transaction Analytics project focused on ETL pipelines, fraud analysis, SQL database modeling, and interactive Power BI dashboards.

---

## 📌 Project Overview

This project simulates a real-world banking analytics environment where transaction data is processed, cleaned, stored in MySQL, and visualized using Power BI for business insights and fraud monitoring.

The system was built to understand how raw financial transaction data flows through:
- Data preprocessing
- ETL pipelines
- Database systems
- Analytics dashboards

---

## ⚡ Features

### 🔹 Data Engineering
- ETL Pipeline using Python
- Automated data cleaning & preprocessing
- Transaction normalization
- Fraud label handling

### 🔹 Database Design
- MySQL relational database integration
- Normalized schema design
- One-to-many relationship modeling:
  - One customer → multiple accounts
  - One account → multiple transactions

### 🔹 Analytics & Visualization
- Interactive Power BI dashboard
- Fraud monitoring KPIs
- Transaction trend analysis
- Account type distribution
- Transaction type insights
- Dynamic slicers & filters
